# ü§ù Contributing to the Ultimate AI Tools Directory

Thank you for your interest in contributing to the **Ultimate AI Tools Directory**! Your help is essential to keeping this resource comprehensive, accurate, and up-to-date.

This guide outlines the process for suggesting new tools, updating existing information, and contributing to the repository's structure and documentation.

---

## üí° How to Suggest a New AI Tool

We welcome submissions of new, innovative AI tools. Please follow these steps to ensure your submission is processed quickly:

### 1. Check for Duplicates

Before submitting, please use the GitHub search bar to check if the tool is already listed in any of the [Category READMEs](./categories/).

### 2. Open a New Issue

If the tool is not listed, open a new issue using the **"Suggest a New AI Tool"** template.

**Required Information:**

| Field | Description | Example |
| :--- | :--- | :--- |
| **Tool Name** | The official name of the AI tool. | `Midjourney` |
| **Official Website URL** | The direct link to the tool's homepage. | `https://www.midjourney.com/` |
| **Category** | Select the best-fitting category from the [Category Index](../README.md#%F0%9F%97%82%EF%B8%8F-category-index). | `Image Generation` |
| **Description** | A concise, one-sentence summary of what the tool does. | `An independent research lab that produces a proprietary artificial intelligence program to create images from textual descriptions.` |
| **Pricing Model** | State the pricing model clearly. | `Freemium (Free tier available, paid subscription for full features)` |

### 3. Review Process

A maintainer will review your submission. If approved, the tool will be added to the appropriate category file and the issue will be closed.

**Required Information:**

| Field | Description |
| :--- | :--- |
| **Tool Name** | The name of the tool with the error. |
| **Category** | The category where the tool is currently listed. |
| **Nature of Error** | Briefly describe the issue (e.g., "Broken link," "Outdated description," "Incorrect pricing"). |
| **Correction** | Provide the correct information (e.g., the new URL, the updated description, or the new pricing model). |

## üåê Contributing to Translations

The repository aims to be fully multilingual. If you are a native speaker of one of the placeholder languages (Arabic, Chinese, Hindi, Spanish, French, etc.), your contribution is invaluable!

### 1. Translate the Tool Data

We need help translating the `name` and `description` fields for all tools. This is a large task, so please contact the maintainer on Telegram to coordinate the effort: **@saddex.x**

### 2. Translate Documentation

You can submit a Pull Request to translate the main `README.md` or this `CONTRIBUTING.md` file into your native language.

## ‚öôÔ∏è Repository Maintenance (For Advanced Contributors)

If you wish to contribute directly to the code or repository structure, please follow the standard GitHub workflow:

1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  Create a new **branch** for your changes (`git checkout -b feature/my-new-feature`).
4.  Make your changes and **commit** them (`git commit -m "feat: add new feature"`).
5.  **Push** your branch (`git push origin feature/my-new-feature`).
6.  Open a **Pull Request** against the `main` branch of the original repository.

All Pull Requests will be reviewed before merging. Thank you for making this the ultimate resource!
